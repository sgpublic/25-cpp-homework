# 教材习题7_10：定义一个Object类和Box类

## 问题描述

定义一个 Object 类，有数据成员 weight 及相应的操作函数，由此派生出 Box 类，增加数据成员 height 和 width 及相应的操作函数，声明一个 Box 对象，观察构造函数和析构函数的调用顺序。

## 输入输出形式

参考的输入及输出（其中Input开头的两段文字为输入提示和输入数据，其他均为输出）：

```
Constructing Object!
Constructing Box!
Input weight:50
Input height and width:20 30
Weight:50
Height and width:20,30
Destructing Box!
Destructing Object!
```

# 利用对象数组，计算10个顶点组成折线的长度

## 问题描述

假设有一个点类 point，具有两个实数坐标。希望主程序使用这个类完成下述功能：
1. 主程序为类 point 定义 10 个点的对象数组（也可以动态申请空间）。
2. 要求调用一个函数 Set() 从键盘输入10个对象的属性。
3. 要求调用一个函数 Display() 显示10个对象的值。
4. 要求调用一个函数 Lenth()，计算将这些点连成一条折线时，这条折线的长度。

## 输入形式

参考的输入（每个点要求提示“Input x,y:”）：

```
Input x,y:0 0
Input x,y:3 5
Input x,y:2 4
Input x,y:9 8
Input x,y:2 3
Input x,y:8 9
Input x,y:7 6
Input x,y:2 9
Input x,y:5 8
Input x,y:6 6
```

## 输出形式

参考的输出：

```
(0,0)
(3,5)
(2,4)
(9,8)
(2,3)
(8,9)
(7,6)
(2,9)
(5,8)
(6,6)
Length:46.7866
```
